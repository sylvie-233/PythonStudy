## 基础介绍


轻量级WSGI框架 


Flask默认5000端口


flask_sqlalchemy只提供Engine、Model、Session，表字段定义仍为原框架SQLAlchemy

flask_login提供登录管理LoginManager、用户模型UserMixin

flask_wtf提供了表单页面、表单模型类、表单字段校验



### 开发目录结构
```yaml
flask:
    /models:
    /routes:
    /services:
    /static:
    /templates:
    app.py:
```







### flask
```yaml
flask:
    --app: 指定主模块
    --debug: 开发模式
    --host: 指定ip（公网ip：0.0.0.0）
    run: 运行
```
