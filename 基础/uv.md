# uv



## 基础介绍

uv 是一个现代的 Python 包管理器，由 Astral 开发，目标是 比 pip 和 virtualenv 更快、更安全、更现代化
兼容 requirements.txt 和 pyproject.toml


### uv
```yaml
uv:
    --version:
    add: # 添加依赖
    init: # 新建项目脚手架
    pip:
        compile:
        install:
        sync:
    remove: # 移除依赖
    tool: # uv 工具链
        install:
    tree: # 依赖树
    venv: # 创建虚拟环境
```
