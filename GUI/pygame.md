# pygame



## 基础介绍

pygame 是一个简单的游戏开发库，可以帮助你创建 2D 游戏、图形、音效等
`uv add pygame`

## 核心内容
```yaml
pygame:
    display: # 窗口
        flip():
        set_caption():
        set_mode():
            fill():
    draw: # 绘制
        circle():
        rect():
    event: # 事件
        get():
    font: # 字体
    image: # 图片
        load():
    mixer: # 音频
        Sound:
            play():
        init(): 
    time: # 时间
        Clock:
            tick():
    QUIT:
    init():
    quit():
```
