# tinydb



## 基础介绍

轻量级的 本地 JSON 数据库
`uv add tinydb`


## 核心内容
```yaml
tinydb:
    middlewares:
        CachingMiddleware:
        MemoryStorage:
    operations:
        set():
    storages:
        JSONStorage:
    Query:
        matches():
    TinyDB:
        all():
        get():
        insert():
        remove():
        search():
        table():
            insert():
        truncate():
        update():
```
