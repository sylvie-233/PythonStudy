# python-docx



## 基础介绍

操作 Word 文档库
支持创建、读取、修改 .docx 文件（MS Word 2007及以上版本的格式）
`uv add python-docx`

## 核心内容
```yaml
docx:
    Document:
        paragraphs: 
            text:
        add_heading():
        add_paragraph():
        add_picture():
        add_run():
        add_table():
        save():
```
