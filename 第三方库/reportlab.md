# reportlab

## 基础介绍

Python 创建 PDF 文档的库

## 核心内容
```yaml
reportlab:
    lib:
        colors:
        pagesizes:
            A4:
        styles:
            getSampleStyleSheet():
    pdfgen:
        canvas:
            Canvas:
                circle():
                drawImage():
                drawString():
                line():
                rect(): 
                save(): # 保存
                setFillColor():
                setFillColorRGB():
                setFont():
                setLineWidth():
                setStrokeColor():
    platypus:
        Frame:
        Image:
        PageTemplate():
        Paragraph:
        SimpleDocTemplate:
            build():
        Spacer:
        Table:
        TableStyle:
```
